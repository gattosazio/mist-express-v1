# IT and Data Security

Department: Information Technology
Policy Type: it_data_security
Version: v1
Effective Date: 2026-04-26

## Access and Authentication

Employees must use company-issued credentials to access company systems.
Multi-factor authentication is required for email, VPN, cloud administration tools, and systems that store sensitive company information.

## Device Security

Company-issued devices must be protected with approved security controls, including screen lock, endpoint protection, and current security updates.
Employees may not disable security tools installed by Information Technology.

## Data Handling

Sensitive company data must be stored only in approved company systems.
Employees may not transfer sensitive company data to personal storage, personal email, or unapproved third-party services.

## Incident Reporting

Suspected phishing, credential compromise, malware infection, or unauthorized access must be reported to the Information Security team immediately after discovery.
