# Sample Company FAQ Corpus

These files form a single sample company handbook corpus for production-style RAG testing.

## Recommended upload values

Use the values written at the top of each file when filling your form:

- `title`: file title
- `department`: `Department:` value
- `policy_type`: `Policy Type:` value
- `version`: `Version:` value
- `effective_date`: `Effective Date:` value
- `status`: `active`

- How long is the probationary period?
- When is payroll processed?
- Are probationary employees eligible for allowances?
- Who approves planned leave for more than three business days?
- Is MFA required for company systems?
